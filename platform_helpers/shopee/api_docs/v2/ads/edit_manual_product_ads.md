More

| --- | --- | --- | --- |
| partner\_id | int | 1 | Partner ID is assigned upon registration is successful. Required for all requests. |
| timestamp | timestamp | 1610000000 | This is to indicate the timestamp of the request. Required for all requests. Expires in 5 minutes. |
| access\_token | string | c09222e3fc40ffb25fc947f738b1abf1 | The token for API access, using to identify your permission to the api. Valid for multiple use and expires in 4 hours. |
| shop\_id | int | 600000 | Shopee's unique identifier for a shop. Required param for most APIs. |
| sign | string | e318d3e932719916a9f9ebb57e2011961bd47abfa54a36e040d050d8931596e2 | Signature generated by partner\_id, api path, timestamp, access\_token, shop\_id and partner\_key via HMAC-SHA256 hashing algorithm. ## Request Parameters

Collapse


| Name | Type | Required | Sample | Description |
| --- | --- | --- | --- | --- |
| reference\_id | string | True | asdf-1234-qwerty | A random string used to prevent duplicate ads. If an ads is created successfully, subsequent request using the same reference id will fail |
| campaign\_id | int64 | True | 112234 | The unique identifier for a campaign |
| edit\_action | string | True | start | Actions supported: <br>"start", "pause", "resume", "stop", "delete", "change\_budget", "change\_duration", "change\_smart\_creative", "change\_location", "change\_enhanced\_cpc", "change\_roas\_target" |
| budget | float | False | 10.50 | The budget set for the Auto Product Ads |
| start\_date | string | False | 11-11-2023 | the start date per campaign. please kindly note that if you want to set unlimited date, you can just pass today's date as the start date |
| end\_date | string | False |  | the end date of each campaign. please kindly note that if you want to set an unlimited campaign, you can keep empty for the end date field |
| roas\_target | float | False | 1.20 | the ROAS target for each campaign with auto bidding |
| discovery\_ads\_locations | object\[\] | False |  | the location settings for manual bidding method |
| location | string | True | daily\_discover | daily\_discover, you\_may\_also\_like |
| status | string | True | active | active / inactive |
| bid\_price | float | True | 10.50 | bid price of the location |
| enhanced\_cpc | boolean | False | true | Enhanced CPC functionality toggle |
| smart\_creative\_setting | string | False | default | Whether to use default or set on/off Smart Creative for this ad. Supported Values: "", "default", "on", "off". Empty string treated as default. |
